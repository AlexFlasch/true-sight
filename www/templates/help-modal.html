Finding Your Vanity Name
Log into Steam and navigate to your profile page.
In the URL you should see one of the following:
  • http://steamcommunity.com/id/[yourVanityName]
  • http://steamcommunity.com/profiles/7656119xxxxxxxxxx

For example: In this example you would only input ExampleVanityName
It would work the same way for the second example:
Just input the number after the /profiles/
